Sosamba<br>

A Discord framework that uses Eris to power tt.bot v4, usable in other projects as well.

Installation

Currently this framework is available only from GitHub.
In order to use Sosamba, you need node.js >=10.0.0. ```sh

if using NPM

npm i github:tt-bot-dev/sosamba --save

if using Yarn

yarn add github:tt-bot-dev/sosamba ```

Projects using Sosamba

Take a look at the sosamba topic. To add your own, simply add that topic to your repository.

How to use Sosamba

You can look at the documentation here, or look at tt.bot (currently on the v4-wip branch).
